project:

IntergalacticDB

for whom was it done:

Software Engineering Class

what was done:

Star Wars Universe database web application.

when was it done:

Summer 2015

where was it done:

GDC@utexas | Computer Lab

why was it done:

Because of the extensive amount of Star Wars data online, we wanted to make a simple Star Wars database allowing easy access to the information about the main characters, planets, and species. This is our 3-phase project for Software Engineering class. It helps us learn different tools for web application development.

how was it done:

Tools: Github, Python 3.4, Flask, SQLAlchemy, Swiftype, Rackspace, nginx, gunicorn, supervisorctl, coverage, unittest
