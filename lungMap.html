project:

bisque-spreadsheet

for whom was it done:

Dr. James Carson

Center for Lung Development Imaging and Omics

Life Science Computing group at Texas Advance Computing Center (TACC)

what was done:

Python script to upload a spreadsheet (tab-delimited) of multiple metadata (annotations/tags) to BisQue Image Analysis Environment. Each row of the spreadsheet will be an image and each column will be an annotation. In addition, the user can also download metadata for all/query-specific images into a spreadsheet format.

when was it done:

May 2015

where was it done:

My apartment, since majority of the time I worked from home with additional meeting with the supervisor.

why was it done:

LungMap project has an enormous amount of images and metadata to maintain. The current version of Bisque only allows web interface metadata upload one at a time.

how was it done:

Tools: Python 2.7 for scripting, Bisque's API with the knowledge of RESTful for accessing the web interface data, TACC's Lonestar machine to maintain and store data
